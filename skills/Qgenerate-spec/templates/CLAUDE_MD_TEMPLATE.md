### 작업 상태
| 상태 | 의미 |
|------|------|
| 🔲 진행 전 | 아직 시작하지 않은 작업 |
| 🔶 진행 중 | 현재 작업 중 |
| ✅ 완료 | VERIFY_CHECKLIST의 모든 항목 체크됨. **더 이상 참조 불필요.** |

- VERIFY_CHECKLIST의 **모든 체크박스가 체크**되면 ✅ 완료
- 완료된 작업 파일은 **참조하지 않아도 됩니다.**
